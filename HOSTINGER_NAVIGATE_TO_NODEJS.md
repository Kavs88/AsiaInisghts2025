# Navigate to Node.js Web Apps - Step by Step

## 🎯 You're on the Wrong Page

The page you're seeing is for **regular websites** (WordPress, static sites). For your **Next.js app**, you need the **Node.js Web Apps** section.

---

## ✅ Correct Navigation Path

### Step 1: Go to Node.js Web Apps

1. **In hPanel**, look for one of these:
   - **"Websites"** in the left sidebar
   - **"Node.js Apps"** or **"Node.js Web Apps"** in the left sidebar
   - **"Add Website"** button → Then select **"Node.js Apps"**

2. **Alternative paths:**
   - hPanel → **"Websites"** → **"Add Website"** → Select **"Node.js Apps"**
   - hPanel → **"Advanced"** → **"Node.js"** (older interface)
   - hPanel → **"Node.js Web Apps"** (newer interface)

### Step 2: Create New Node.js App

1. Click **"Create Node.js App"** or **"Add Website"** → **"Node.js Apps"**
2. You should see options like:
   - **"Upload your website files"**
   - **"Deploy from GitHub"**

### Step 3: Upload Your Zip

1. Select **"Upload your website files"**
2. Upload: `hostinger-deploy-clean.zip`
3. Wait for upload to complete

### Step 4: Configure Settings

After upload, you'll see the build configuration page:
- ✅ Framework: Next.js (auto-detected)
- ✅ Node version: 20.x
- ✅ Root directory: ./
- ✅ Build settings: Default
- ✅ **Add environment variables** (5 variables)

---

## 🔍 If You Can't Find Node.js Section

### Check Your Hosting Plan

Node.js apps require:
- ✅ **Business Web Hosting**
- ✅ **Cloud Startup**
- ✅ **Cloud Professional**
- ✅ **Cloud Enterprise**
- ✅ **Cloud Enterprise Plus**

**NOT supported on:**
- ❌ Shared hosting plans
- ❌ Basic hosting plans

### If Node.js Section is Missing

1. **Check your hosting plan:**
   - Go to: hPanel → **"Account"** or **"Billing"**
   - Verify you have a plan that supports Node.js

2. **Upgrade if needed:**
   - Contact Hostinger support
   - Upgrade to a Business or Cloud plan
